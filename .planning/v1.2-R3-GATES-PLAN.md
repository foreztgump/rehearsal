# R3 Operator-Gate Enablement Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make R3's operator gates G1–G5 actually runnable by building, in the sandbox, every gate-blocking piece that does not require a GPU — the Parakeet engine body, the int8 weight bake, and a reproducible WER/cut-off harness — so the RTX 5090 session is purely build → run → record.

**Architecture:** `stt/backend_parakeet.py` becomes a thin wrapper over the canonical sherpa-onnx `OfflineRecognizer` (D22, supersedes D17); the prebuilt int8 bundle is fetched + baked into both STT images; a `scripts/stt-wer.py` harness reuses that same engine over committed CC-BY fixtures to produce real WER on CPU; `vram-validate.sh` gains an arithmetic helper to derive the hybrid VRAM threshold; and `v1.2-R3-VERIFY.md` is rewritten as a deterministic step-tagged runbook.

**Tech Stack:** Python 3.11, sherpa-onnx (k2-fsa), jiwer, onnxruntime/NeMo (existing), bash, Docker, the project's `__main__` assert-harness test convention (no pytest).

**Spec:** `.planning/v1.2-R3-GATES-DESIGN.md`. **Inherits:** `.planning/v1.2-R3-DESIGN.md` (D13–D20).

## Global Constraints

- **Pin every version; never `:latest`.** `sherpa-onnx~=1.13` (PyPI 1.13.3), `jiwer~=4.0` (PyPI 4.0.0). Buffered model = **`sherpa-onnx-nemo-parakeet-tdt-0.6b-v2-int8`** (v2, English — **NOT v3**, which regresses English). License: model CC-BY-4.0, fixtures LibriSpeech CC-BY-4.0 — attribution required.
- **`stt/backend_parakeet.py` MUST byte-import with NO sherpa-onnx / onnxruntime installed** — every heavy import (`sherpa_onnx`, `numpy`) lives INSIDE `load_model`/`finalize`. `stt/test_engine.py::_assert_parakeet_imports_without_ort` enforces this.
- **No magic values** — named constants for filenames, sample rate (`16000`), feature dim (`80`), int16 full-scale (`32768.0`), the WER ceiling. (Latency budgets stay named constants per CODE_PRINCIPLES.)
- **Tests are `__main__` assert harnesses** run via `python3 <path>`, printing `"<name> OK"` to stderr, exit 0 on pass. No pytest, no fixtures framework (matches `stt/test_engine.py`, `tests/test_placement.py`).
- **Local-first:** weights/fixtures are fetched at build/setup and run offline after. Committed audio is **inbound public CC-BY** data (not user audio leaving the LAN); the operator's own-voice cut-off clip stays **uncommitted**.
- **Function shape:** ≤40 lines, ≤3 params, ≤3 nesting levels; SRP.
- **Streaming is untouched.** No change to `STT_ENGINE=streaming` behaviour (G4 guards byte-identity). Do not edit `backend_nemo.py`, `backend_onnx.py`, or the streaming path of `server.py`.
- **STT image build context is `./stt`** (both services in `docker-compose.yml`). Anything a Dockerfile `COPY`s must live under `stt/`. (This is why the fetch script lives at `stt/fetch_parakeet_onnx.sh`, not `scripts/` — a deviation from the spec's touchpoint list, justified by build reachability.)
- **`[network, one-time]`** marks steps that download large assets (the 622 MB bundle, LibriSpeech). The CODE is offline-buildable; only this DATA needs network once. If the sandbox lacks network, those steps run on the GPU host and the dependent self-check degrades to pure-logic asserts (spec risk #3) — never silently skipped.

```bash
#!/usr/bin/env bash
#
# fetch_parakeet_onnx.sh — download + verify the prebuilt sherpa-onnx Parakeet-tdt-0.6b-v2
# int8 ONNX bundle (R3 buffered/hybrid `final` backend; D22). Pinned to the k2-fsa
# `asr-models` release asset (a fixed tag, never :latest); the bundle sha256 is recorded
# in .planning/research/STACK.md for provenance. Build-time download, offline after
# (local-first; mirrors stt/Dockerfile's snapshot-then-offline .nemo bake).
#
# Usage: stt/fetch_parakeet_onnx.sh <target_dir>
#   <target_dir> receives encoder.int8.onnx / decoder.int8.onnx / joiner.int8.onnx / tokens.txt
set -euo pipefail

readonly BUNDLE_URL="https://github.com/k2-fsa/sherpa-onnx/releases/download/asr-models/sherpa-onnx-nemo-parakeet-tdt-0.6b-v2-int8.tar.bz2"
readonly REQUIRED_FILES=(encoder.int8.onnx decoder.int8.onnx joiner.int8.onnx tokens.txt)

fail() { echo "fetch_parakeet_onnx: $*" >&2; exit 1; }

verify_bundle() {  # <dir> — assert all four model files exist and are non-empty
  local dir="$1" name
  for name in "${REQUIRED_FILES[@]}"; do
    [ -s "${dir}/${name}" ] || fail "missing/empty ${dir}/${name}"
  done
}

main() {
  [ "$#" -eq 1 ] || fail "usage: fetch_parakeet_onnx.sh <target_dir>"
  local target="$1" tmp
  mkdir -p "${target}"
  tmp="$(mktemp)"; trap 'rm -f "${tmp}"' EXIT
  echo "fetch_parakeet_onnx: downloading pinned bundle..." >&2
  curl -fsSL "${BUNDLE_URL}" -o "${tmp}"
  tar -xj -C "${target}" --strip-components=1 -f "${tmp}"
  verify_bundle "${target}"
  echo "fetch_parakeet_onnx: OK -> ${target}" >&2
}

# Sourceable for testing: only run main when executed directly.
if [ "${BASH_SOURCE[0]}" = "${0}" ]; then
  main "$@"
fi
```

```python
    """Transcribe the whole accumulated turn buffer once (offline; native PnC as-is)."""
    pcm = bytes(state.get("_turn_pcm", b""))
    if not pcm:
```

- [ ] **Step 2: Write the harness**

Create `scripts/stt-wer.py`:

```python
#!/usr/bin/env python3
"""stt-wer.py — R3 G1/G5 accuracy harness: WER + trailing-word cut-off check.

Runs the REAL buffered Parakeet engine (stt/backend_parakeet via sherpa-onnx) over the
committed fixtures and reports WER vs ground truth; also flags end-of-speech word-drop.

  python3 scripts/stt-wer.py                        # aggregate WER over the fixture set
  python3 scripts/stt-wer.py --cut-off <wav> <ref>  # trailing-drop flag for one clip
  python3 scripts/stt-wer.py --self-check           # pure-logic asserts (+ real clip if deps present)

Deps (host/dev only — never in an image): pip install -r requirements-dev.txt
"""
from __future__ import annotations

import json
import os
import re
import sys
import wave

_REPO_ROOT = os.path.dirname(os.path.dirname(os.path.abspath(__file__)))
_FIXTURE_DIR = os.path.join(_REPO_ROOT, "tests", "fixtures", "asr")
_TRANSCRIPTS = os.path.join(_FIXTURE_DIR, "transcripts.jsonl")
_CUTOFF_TAIL_WORDS = 3
_SELF_CHECK_WER_CEILING = 0.15
_PUNCT = re.compile(r"[^\w\s]")


def _normalize(text: str) -> str:
    """Lowercase + strip punctuation so Parakeet PnC compares fairly to plain refs."""
    return _PUNCT.sub("", text.lower()).strip()


def read_wav_pcm(path: str) -> bytes:
    """Raw int16 mono PCM bytes from a 16 kHz WAV (stdlib only — no soundfile dep)."""
    with wave.open(path, "rb") as handle:
        if handle.getnchannels() != 1 or handle.getsampwidth() != 2:
            raise ValueError(f"{path}: expected 16-bit mono WAV")
        return handle.readframes(handle.getnframes())


def transcribe(pcm: bytes) -> str:
    """Run the real buffered Parakeet engine over PCM (reuses stt/backend_parakeet)."""
    sys.path.insert(0, os.path.join(_REPO_ROOT, "stt"))
    import backend_parakeet as engine
    model = engine.load_model()
    state = engine.new_stream_state(model)
    state["_turn_pcm"] = bytearray(pcm)
    return engine.finalize(model, state)


def word_error_rate(reference: str, hypothesis: str) -> float:
    import jiwer
    return jiwer.wer(_normalize(reference), _normalize(hypothesis))


def trailing_drop(reference: str, hypothesis: str, tail_words: int = _CUTOFF_TAIL_WORDS) -> bool:
    """True if the hypothesis appears to drop trailing words vs the reference tail."""
    ref, hyp = _normalize(reference).split(), _normalize(hypothesis).split()
    if len(ref) <= tail_words:
        return False
    return len(hyp) < len(ref) and ref[-tail_words:] != hyp[-tail_words:]


def load_fixtures() -> list:
    with open(_TRANSCRIPTS, encoding="utf-8") as handle:
        return [json.loads(line) for line in handle if line.strip()]


def run_aggregate() -> int:
    import jiwer
    fixtures = load_fixtures()
    references, hypotheses = [], []
    for fixture in fixtures:
        pcm = read_wav_pcm(os.path.join(_FIXTURE_DIR, fixture["clip"]))
        hypothesis = transcribe(pcm)
        references.append(_normalize(fixture["text"]))
        hypotheses.append(_normalize(hypothesis))
        print(f"{fixture['clip']}: WER={jiwer.wer(_normalize(fixture['text']), _normalize(hypothesis)):.3f} "
              f"drop={trailing_drop(fixture['text'], hypothesis)}")
    print(f"AGGREGATE int8 Parakeet WER = {jiwer.wer(references, hypotheses):.4f} over {len(references)} clips")
    return 0


def run_cutoff(wav_path: str, reference: str) -> int:
    hypothesis = transcribe(read_wav_pcm(wav_path))
    print(f"hypothesis: {hypothesis}\ntrailing_drop={trailing_drop(reference, hypothesis)}")
    return 0


def _self_check() -> int:
    # Pure-logic asserts (no audio/model) — always run.
    assert word_error_rate("the cat sat", "the cat sat") == 0.0
    assert abs(word_error_rate("the cat sat on the mat", "the cat sat on the") - 1 / 6) < 1e-9
    assert trailing_drop("the quick brown fox jumps", "the quick brown fox") is True
    assert trailing_drop("the quick brown fox jumps", "the quick brown fox jumps") is False
    assert trailing_drop("hi there", "hi") is False
    print("stt-wer pure-logic OK (wer + trailing_drop)", file=sys.stderr)
    # Real transcribe — best-effort (needs sherpa-onnx + fetched weights + fixtures).
    try:
        fixtures = load_fixtures()
        hypothesis = transcribe(read_wav_pcm(os.path.join(_FIXTURE_DIR, fixtures[0]["clip"])))
        wer = word_error_rate(fixtures[0]["text"], hypothesis)
        assert wer < _SELF_CHECK_WER_CEILING, f"WER {wer:.3f} >= ceiling {_SELF_CHECK_WER_CEILING}"
        print(f"stt-wer real-clip OK: WER={wer:.3f} ('{hypothesis}')", file=sys.stderr)
    except (ImportError, FileNotFoundError, OSError, ValueError, SystemExit, IndexError) as exc:
        print(f"stt-wer real-clip SKIPPED (GPU-deferred): {type(exc).__name__}: {exc}", file=sys.stderr)
    return 0


def main(argv: list) -> int:
    if argv and argv[0] == "--self-check":
        return _self_check()
    if argv and argv[0] == "--cut-off":
        if len(argv) != 3:
```

```bash
# G2 (R3): derive Parakeet's GPU footprint + the recommended VRAM_HYBRID_MB threshold from
# two measured peaks (hybrid vs streaming). Pure arithmetic — no GPU. The 12 GB boundary is
# COMPUTED, not physically tested (only a 24 GB card is available; D24).
derive_hybrid_threshold() {  # <hybrid_peak_mb> <streaming_peak_mb>
  local hybrid_peak="$1" streaming_peak="$2"
  local parakeet_mb=$((hybrid_peak - streaming_peak))
  local min_total=$((hybrid_peak + VRAM_HEADROOM_MB))
  local recommended=$(( (min_total + 1023) / 1024 * 1024 ))   # round up to whole GB
  echo "parakeet_gpu_mb=${parakeet_mb} min_total_mb=${min_total} recommended_VRAM_HYBRID_MB=${recommended}"
}

self_test() {  # sandbox arithmetic check — no GPU
  local out; out="$(derive_hybrid_threshold 14000 11600)"
  [ "${out}" = "parakeet_gpu_mb=2400 min_total_mb=15024 recommended_VRAM_HYBRID_MB=15360" ] \
    || fail "derive_hybrid_threshold self-test mismatch: ${out}"
  echo "vram-validate self-test OK (derive_hybrid_threshold)" >&2
}
```
