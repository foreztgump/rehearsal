# v1.2 R3 — Plan A: Core STT Engine Seam + Buffered Parakeet — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn the STT sidecar's single-backend pipeline into a 3-mode `STT_ENGINE` selector (`streaming` / `buffered` / `hybrid`) by composing a `primary` backend (live partials + endpointing) with an optional `final` backend (authoritative transcript), and add a buffered Parakeet-int8-ONNX backend that eliminates the 15a end-of-speech word-drop.

**Architecture:** Server-side composition over the existing 5-callable seam (`load_model / new_stream_state / decode_chunk / finalize / reset_turn_state`). A backend declares a `STREAMS: bool` capability; the server emits deltas + text-stall EOU when the *primary* streams, else accumulates raw PCM + uses an RMS energy-EOU; `finalize` always routes to the *final* backend. The WS contract (`ready`/`delta`/`final`/`error`) is byte-unchanged, so the agent plugin barely changes.

**Tech Stack:** Python (uv), FastAPI websocket server, `onnxruntime` / `onnxruntime-gpu` (buffered Parakeet), the existing NeMo streaming sidecar, `livekit-agents` 1.6.x plugin.

**Scope:** This is **Plan A of 3** for R3. Plan B (hardware awareness: `placement.py` VRAM-derive + `resolve_parakeet_device` + `gpu-doctor.sh` advise mapping) and Plan C (`TTS_ENGINE` seam + `finalize()` runbook + docs reframe + `v1.2-R3-VERIFY.md`) follow once Plan A's interfaces are locked. Full design: `.planning/v1.2-R3-DESIGN.md`.

## Global Constraints

- `STT_ENGINE ∈ {streaming, buffered, hybrid}`, **default `streaming`** (byte-for-byte back-compat). Invalid → `SystemExit` (mirrors `STT_RUNTIME`).
- `STT_BUFFERED_DEVICE ∈ {cpu, gpu}`, **default `cpu`** (selects the Parakeet onnxruntime EP). Plan B adds auto-resolution; Plan A reads it directly.
- Buffered engine = **`nvidia/parakeet-tdt-0.6b-v2`** int8 ONNX. **Pin v2, NOT v3** (v3 regresses English). `nemo-toolkit` 2.7.3-class for any NeMo export; license **CC-BY-4.0** (attribution).
- Buffered backend declares **`STREAMS = False`**; streaming backends declare `STREAMS = True`.
- Energy-EOU reuses the existing **`ENDPOINT_SILENCE_MS`** (default 700) for the silence window; RMS threshold is a **named constant** `ENERGY_SILENCE_RMS` (no magic numbers).
- Max-buffer cap: **`STT_BUFFERED_MAX_S`** (default 60) bounds `state["_turn_pcm"]`; on cap → force-finalize.
- **Heavy imports (`onnxruntime`, `nemo`, `numpy`) live INSIDE backend functions** so every module byte-compiles in the GPU-less, ORT-less sandbox (the existing `backend_nemo`/`backend_onnx` rule).
- **No hardcoded model tags** — single-source via env, `KeyError → SystemExit` posture.
- Tests are **stdlib `assert`-based `_self_check()` run via `python3 <file>`** (matching `stt/test_dispatch.py` / `agent/placement.py`) — **NOT pytest**. AAA structure.
- Code rules: ≤40 lines/function, ≤3 params, ≤3 nesting, error handling on every boundary, named constants only.
- Energy-EOU uses **stdlib `array` + `math`** for RMS (no numpy in `server.py`; keeps the sandbox byte-compile + avoids the deprecated `audioop`).
- Local-first: no cloud endpoints; weights bake offline into the image.

```python
# R3 buffered/energy-EOU (no numpy: stdlib array+math keeps server.py sandbox-safe).
# When the PRIMARY backend has STREAMS=False there is no growing transcript to
# stall-detect, so end-of-utterance is detected acoustically: chunks whose RMS sits
# below ENERGY_SILENCE_RMS for ENDPOINT_SILENCE_MS (after voiced audio) end the turn.
ENERGY_SILENCE_RMS = float(os.environ.get("STT_ENERGY_SILENCE_RMS", "320"))  # int16 RMS floor
# Bound the per-turn PCM buffer so a missed EOU cannot grow memory unbounded.
BUFFERED_MAX_S = int(os.environ.get("STT_BUFFERED_MAX_S", "60"))


def rms_int16(pcm: bytes) -> float:
    """RMS of int16 mono PCM via stdlib array+math (no numpy, no deprecated audioop)."""
    import array
    import math
    samples = array.array("h")
    samples.frombytes(pcm[: len(pcm) - (len(pcm) % 2)])
    if not samples:
        return 0.0
    return math.sqrt(sum(s * s for s in samples) / len(samples))
```

    """No partials. Accumulate PCM; fire ONE final after ENDPOINT silence of low RMS
    following voiced audio, or when the max-buffer cap is hit."""
