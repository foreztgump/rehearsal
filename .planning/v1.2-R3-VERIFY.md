R3 is closed by operator/user acceptance after a full-stack voice-to-voice smoke test on
the target stack. Exact latency, VRAM, and candidate A/B numbers were not captured, so the

## Closeout Summary

- **Result:** PASS / closed 2026-06-29 by user acceptance.
- **Final selected path:** `STT_ENGINE=buffered` with `STT_MODEL=nvidia/parakeet-tdt-0.6b-v2`.
  Good NVIDIA hardware should run the GPU service (`--profile stt-gpu`,
  `STT_FORCE_CPU=0`, `STT_HEADROOM_MEASURED=1`); CPU remains the VRAM-safe fallback.
- **Full-stack smoke:** mic -> STT -> agent/LLM -> TTS -> audio back worked end to end.
- **Observed STT result:** GPU `buffered` was much more accurate than `streaming` with acceptable latency.
- **Accepted latency tune:** `STT_STREAM_CHUNK_MS=320`, `STT_ENDPOINT_SILENCE_MS=640`.
- **Barge-in:** `INTERRUPT_MIN_DURATION_S=0.25` felt sensitive but usable; leave as-is unless false interrupts show up.
- **Sandbox confirmation:** `stt/test_dispatch.py`, `stt/test_engine.py`, Parakeet bake/fetch checks,
  placement, gpu-doctor, vram self-test, and Parakeet fixture WER all passed.
- **Not captured:** quantitative voice-to-voice p50/p95, GPU buffered VRAM peak under full load, buffered EOU->final p50,
  streaming-vs-buffered WER on the operator audio, and finalize candidate A/B measurements.
